# 这个文件夹放节点照片

用法：
1. 把照片放进这个文件夹，例如 `together.jpg`
2. 打开 `js/data.js`，找到对应节点，把 `photo: ''` 改成 `photo: 'together.jpg'`
3. 刷新网页，照片就会显示在该节点的详情页里

节点 id 对照：
- prologue     我出生（序章）
- birth        你出生
- adult18      十八岁
- gaokao       高考
- bachelor     本科毕业
- meet         我们相识
- together     我们在一起
- master       硕士答辩
- doubledegree 中丹双学位
- job          正式入职
- propose      被求婚
- tenthousand  一万天 / 海底捞

建议：图片压缩到宽度 1080px 左右，体积小加载快。
